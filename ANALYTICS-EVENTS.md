# Umami Analytics Events — selfrules.org

Events tracked via `data-umami-event` attributes (Umami custom events).

## Navigation Events
| Event Name | Element | Location |
|---|---|---|
| `nav-work` | Work link | Navbar |
| `nav-lab` | Lab link | Navbar |
| `nav-notes` | Notes link | Navbar |
| `nav-contact` | Contact link | Navbar |
| `nav-locale-switch` | EN/IT toggle | Navbar |
