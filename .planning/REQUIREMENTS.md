# Requirements: selfrules.org

**Defined:** 2026-03-22
**Core Value:** Un hiring manager che atterra su selfrules.org pensa "questo è uno che sa quello che fa" — competenza comunicata attraverso specificità e risultati, mai self-promotion.

## Out of Scope

Explicitly excluded. Documented to prevent scope creep.

| Feature | Reason |
|---------|--------|
| Contact form | Coach/consultant pattern — mailto: is more personal and IC-appropriate |
| Newsletter signup | Content marketing / guru energy — incompatible with Modello B |
| Testimonials / social proof | Self-promotional — case study numbers are the proof |
| "Hire me" banner | Kills Modello B instantly |
| Animated page transitions | Over-engineered, adds JS weight (Framer Motion 30KB+), reference sites don't use them |
| Chat widget / Calendly embed | Consultant pattern, anti-IC positioning |
| Multi-language URL slugs | Routing complexity for minimal SEO benefit on personal site |
| PWA / service worker | Nobody needs offline access to a portfolio |
| Comments on blog | Spam, moderation overhead, one-directional "notes" not a conversation |
| Search | 6 pages + 1 post, navigation is sufficient |
| Cookie consent banner | Umami is cookieless, adding one signals uncertainty |
