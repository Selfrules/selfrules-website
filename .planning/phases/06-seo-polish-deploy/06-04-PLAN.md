  <instructions>
    1. Go to Vercel Dashboard -> your project -> Settings -> Domains
    2. Add `selfrules.org` as a domain
       - Vercel will provide DNS records to configure (either A record or CNAME)
       - Update your DNS provider (wherever selfrules.org is registered) with the provided records
       - Wait for DNS propagation (usually 1-5 minutes, sometimes up to 48 hours)
    3. Add `www.selfrules.org` as a domain
       - Configure it as a REDIRECT to `selfrules.org` (Vercel offers this option during domain setup)
       - This satisfies DEPL-03
    4. Verify HTTPS:
       - Vercel automatically provisions SSL certificates
       - After DNS propagates, visit https://selfrules.org and confirm HTTPS works
    5. Verify www redirect:
       - Visit https://www.selfrules.org and confirm it redirects to https://selfrules.org

    **Verification commands (run after DNS propagation):**
    ```bash
    curl -I https://selfrules.org          # Should return 200
    curl -I https://www.selfrules.org      # Should return 301/308 redirect to selfrules.org
    ```
  </instructions>
