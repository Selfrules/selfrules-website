## Interaction Contracts

### Navbar Scroll Behavior (LNAV-01, D-14)

| Property | Value |
|----------|-------|
| Default state | `background: transparent`, no backdrop blur, bottom border `1px solid #1A1A1F` |
| Scrolled state | `background: rgba(10, 10, 11, 0.9)`, `backdrop-filter: blur(12px)`, bottom border `1px solid #1A1A1F` |
| Scroll threshold | 50px (Claude's discretion) |
| Transition | `transition: background-color 200ms ease, backdrop-filter 200ms ease` |
| Position | `position: fixed`, `top: 0`, `left: 0`, `right: 0`, `z-index: 50` |

### Navbar Layout (D-15, D-16)

| Viewport | Left | Center | Right |
|----------|------|--------|-------|
| Desktop (>=768px) | SELFRULES wordmark | -- | About, Work, Lab, Notes, Let's talk (accent), IT / EN |
| Mobile (<768px) | Hamburger icon | SELFRULES wordmark | -- |

**Desktop nav link hover:** Text shifts from `#F5F5F0` to `#E8A838`, `transition: color 150ms ease`.

**Active nav link (PLSH-03):** Accent color (#E8A838) on the link matching the current route. Determined by matching `pathname` against link `href`.

### Mobile Menu (LNAV-03, D-01 through D-05)

| Property | Value |
|----------|-------|
| Trigger | Hamburger icon tap (top-left on mobile) |
| Animation | Slide-in from right, `transform: translateX(100%) -> translateX(0)`, `transition: transform 300ms ease` |
| Backdrop | Full viewport, `background: #0A0A0B` (solid, not translucent) |
| Close trigger | [X] button (top-right), Escape key, nav link click |
| Focus trap | Active when menu open. Tab cycles through menu items only. |
| Body scroll | Locked (`overflow: hidden` on `<body>`) when menu open |
| Content order | Nav links stacked vertically (32px, bold) -> 1px divider (#1A1A1F) -> Language toggle (IT / EN, 16px) |
| Layout | Hamburger left, SELFRULES center or beside hamburger, [X] top-right when open |
| Link spacing | 24px vertical gap between links (minimum 44px tap target height) |
| Menu top padding | 48px below close button |

**Hamburger icon:** Three horizontal lines, 24x24px, `stroke: #F5F5F0`, `stroke-width: 2px`. SVG inline or CSS spans.

**Close icon:** X shape, 24x24px, `stroke: #F5F5F0`, `stroke-width: 2px`. Same position as hamburger (top-right per D-04 -- correction: hamburger is left, close is top-right).

**Hamburger-to-X animation style:** Claude's discretion. Recommended: simple cross-fade (opacity swap), not morphing bars. Keep it fast (200ms).

### Language Toggle (LNAV-04, D-06 through D-08)

**Desktop:**
- Rendered as: `IT` separator `EN` (separator is ` / ` in text tertiary color)
- Active locale: `color: #F5F5F0` (text primary)
- Inactive locale: `color: #5A5A5E` (text tertiary), `cursor: pointer`
- Inactive hover: `color: #8A8A8E` (text secondary), `transition: color 150ms ease`
- Click on inactive: navigates to same path in other locale via next-intl

**Mobile:**
- NOT visible in the mobile navbar bar itself (D-07)
- Lives inside mobile menu, below the nav links, after the 1px divider
- Same active/inactive color logic as desktop
- Size: 16px (slightly larger than desktop 14px for tap target)

## Component Inventory

Phase 2 introduces the following components:

| Component | File Path | Server/Client | Props |
|-----------|-----------|---------------|-------|
| `Navbar` | `src/components/layout/Navbar.tsx` | Client | `locale: string`, `currentPath: string` |
| `MobileMenu` | `src/components/layout/MobileMenu.tsx` | Client | `locale: string`, `currentPath: string`, `isOpen: boolean`, `onClose: () => void` |
| `LanguageToggle` | `src/components/layout/LanguageToggle.tsx` | Client | `locale: string`, `currentPath: string`, `variant: 'desktop' \| 'mobile'` |

**Client components rationale:**
- Navbar: scroll event listener, state for mobile menu open/close
- MobileMenu: animation state, focus trap, escape key listener, body scroll lock
- LanguageToggle: click handler for locale switch
- Footer: no interactivity, pure server render
- NotFoundPage: no interactivity, pure server render

## Layout Structure

### Root Layout Integration

The shell wraps all pages in `src/app/[locale]/layout.tsx`:

```
<html lang={locale}>
  <body className="bg-primary text-text-primary font-sans">
    <a href="#main" className="sr-only focus:not-sr-only ...">Skip to content</a>
    <Navbar locale={locale} currentPath={pathname} />
    <main id="main" className="pt-[64px]">  {/* navbar-height offset */}
      {children}
    </main>
    <Footer locale={locale} />
    {/* Umami script */}
  </body>
</html>
```

### Footer Layout (D-09, D-10, D-11, LNAV-05)

**Desktop (>=768px) -- single row:**
```
[1px border-default top separator]
[page-padding horizontal]

Left side:                          Right side:
  Footer bio tagline                  email (mono, accent hover)
                                      LinkedIn  |  GitHub  |  CV (PDF)

[Footer credit centered or left-aligned, text-tertiary, 14px]
```

**Mobile (<768px) -- stacked:**
```
[1px border-default top separator]
[page-padding horizontal]

Footer bio tagline
email (mono, accent hover)
LinkedIn
GitHub
Scarica il CV (PDF) / Download CV (PDF)

Footer credit
```

**Footer vertical padding:** 32px top, 32px bottom.

### 404 Page Layout (D-12, D-13, LNAV-06)

```
[Navbar -- normal fixed]

[Full remaining viewport height, flex centered]
  Display headline: "404"  (optional, or just the copy)
  Body: "This page doesn't exist. But the rest of the site does."
  Link: "-> Back to homepage" / "-> Torna alla homepage"

[Footer -- normal]
```

**Center alignment:** Both horizontal and vertical within the viewport minus navbar height. Use `min-h-[calc(100vh-64px)]` with flexbox centering.

## Analytics Integration (ANLT-01, ANLT-02, D-17)

| Property | Value |
|----------|-------|

```

```

The script `src` points to `/api/umami/script.js` instead of the Umami cloud URL directly.

## Accessibility Requirements (Phase 2 scope)

| Requirement | Implementation |
|-------------|----------------|
| A11Y-01 Skip-to-content | Hidden link before Navbar, visible on focus, jumps to `#main` |
| A11Y-02 Focus styles | `outline: 2px solid #E8A838`, `outline-offset: 2px` on all interactive elements |
| A11Y-04 Semantic HTML | `<header>` for navbar, `<nav>` for navigation, `<main>` for content, `<footer>` for footer |
| LNAV-03 Focus trap | Mobile menu traps focus when open. First focusable: close button. Last focusable: language toggle. Tab wraps. |
| LNAV-03 Escape closes | `keydown` listener for Escape key closes mobile menu |
